# Esolang

This project is a collection of interpreters, compilers, and other tools for various esoteric programming languages.
Most of the programs are written in Rust, or in the target language themselves.
Some future plans include:
- Interpreters for many of the languages on [esolangs.org](https://esolangs.org/wiki/Category:Programming_languages).
- Compilers for many of these same languages.
- Transpilers from more common languages to esoteric ones.
- A wrapper program that will allow esoteric programming languges that support input and output to make system calls. Something like [pipvm](https://github.com/lestrozi/pipevm), perhaps.
